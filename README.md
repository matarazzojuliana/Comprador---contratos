# Comparador de Contratos - Streamlit

Esta app compara un PDF firmado con un Word original, resaltando cambios y eliminaciones, y entrega un resumen de diferencias.

## Cómo usar
1. Entrar a Streamlit Cloud: https://share.streamlit.io
2. Conectar este repositorio
3. Seleccionar el archivo `app.py`
4. Deploy ? esperar a que la app cargue
5. Subir PDF y Word ? descargar DOCX con cambios
6. (Opcional) Ingresar OpenAI API Key para análisis de implicancias

## Archivos
- `app.py`: código de la app
- `requirements.txt`: librerías necesarias
